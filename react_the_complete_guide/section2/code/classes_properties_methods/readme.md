# ES6 Classes, Properties & Methods

Properties are like "variables attached to classes/"objects"

Methods are like "functions attached to classes/"objects"

## ES6

```javascript
class Sample {
    constructor() {
        this.myProperty = 'value';
    }

    myMethod() {

    }
}
```

## ES7

```javascript
class Sample {
    myProperty = 'value';
    myMethod = ()=> {
        // body
    }
}
```
